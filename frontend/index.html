About the Project

This **Drill Core Lithology Classification System** is built to showcase modern, automated geological logging. Manual classification of drill cores requires highly skilled geologists and is time-consuming. By applying Transfer Learning on deep convolutional neural networks, this system assists in instantly identifying stratigraphic formations.

How it Works

1
Preprocessing

Incoming images are normalized and resized to 224×224 pixels to match the input specifications of the CNN backbone.

2
Feature Extraction (ResNet50)

A pretrained ResNet50 model, trained on over 1 million ImageNet images, extracts hierarchical visual features (texture, grain, boundaries).

3
Classification Head

A customized dense multilayer classifier with dropout regularizations computes the probability distribution across 10 rock types using Softmax activation.
