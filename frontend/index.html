Automated Core Logging Platform

CoreLith is an enterprise SaaS dashboard built to accelerate core analysis workflows in mining and exploration geology. It replaces slow, subjective visual logging with deterministic convolutional predictions.

Deep Learning Pipeline

Our model employs Transfer Learning utilizing a 50-layer ResNet network architecture pretrained on the 1,000-class ImageNet dataset. The feature extraction layers of the backbone are kept frozen to prevent gradients from altering base textures, allowing the system to use prelearned shapes, edges, and shades. A customized dense classification head learns to associate these textures with specific drilling formations.

1. Channel-Wise Normalization

Input images are scaled and normalized using standard ImageNet mean subtraction layers directly within the tensor pipeline, avoiding inference preprocessing mismatch bugs.

2. Multi-Scale Feature Extraction

ResNet50 base convolutions extract micro-bedding lines, porous boundaries, mineral spots, and rock grain gradients.

3. Softmax Density Prediction

Output layer evaluates probability densities over the 10 lithological families using a softmax function: \[P(y = i | \mathbf{x}) = \frac{e^{\mathbf{z}_i}}{\sum_{j=1}^{10} e^{\mathbf{z}_j}}\]
